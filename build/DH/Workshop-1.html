We are given these public parameters, lets see if we can find the private key.

p= 104711
g= 104642
g^a= 1975106301245017026503289083852911126512096109792117074551298200678459015168
g^b= 1504368685604858602888119532110545611422351872

Firstly, we need to find out the secret values of a and b, before we can find the secret/shared key.

Notice that p, g, gᵃ mod p and gᵇ mod p is given. To obtain the value of b, which will thereafter help us find the secret key, we need to solve the discrete logarithm problem. This is where given h, where h equals to gᵇ mod p, we have to determine the value of b. With the value of b, one can compute the secret shared key by computing (gᵃ)ᵇ mod p which is equals to gᵃᵇ mod p.

In this example, we are using this online discrete logarithm calculator. Alternatively, you are recommended to try and use SageMath, a powerful Computer Algebra System (CAS) application used for mathematics.

We have found that:
a: 15
b: 9

After this, we can just use the power operator in python to find the secret key like so: 
pow(g, a*b, p)
